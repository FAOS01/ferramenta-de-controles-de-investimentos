# ferramenta-de-controles-de-investimentos
Exercício DIO planilha para controle de investimentos FII's
É uma planilha de exercício proposta pela DIO para realizar cálculos de aplicações financeiras.
Esta planilha é útil para auxiliar pessoas com e sem experiêria de mercado de FII’s, para aplicar em diversos perfis, bem como para avaliar sua rentabilidade no decorrer do tempo.
Basta baixar a planilha e digitar os valores de salário e de quanto podem investir.
